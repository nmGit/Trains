Resources must be moved around the world, from city to city. Raw 
materials can be shared by cities to produced higher value goods.
Those goods go on to be transported to other cities and so on.

Trains are one of the core transportation mechanisms. Trains are 
loaded at stations and they follow tracks to other stations.
