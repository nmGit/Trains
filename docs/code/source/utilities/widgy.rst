=====
Widgy
=====

.. toctree::
   :maxdepth: 2
   
   Lifecycle <widgy/widget_lifecycle.rst>

Widgy is the utility library that provides traditional user interface elements and mechanisms.

It includes:

* Widgets
 * Buttons
 * Labels
 * etc...
* Layouts
 * Vertical Layouts
 * Horizontal Layouts
 * Grid Layouts
 * etc...
